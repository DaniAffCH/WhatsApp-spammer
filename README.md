# WhatsaApp-spammer
A simple python script that spams a message to a contact as many times as you want

### Prerequisites
* Python3
* pip3
* Selenium
* [GeckoDriver](https://github.com/mozilla/geckodriver/releases)
* Mozilla Firefox

### Installing
Install dependecies
```shell
pip3 install -r requirements.txt
```

Move GeckoDriver to PATH

Run it
```shell
python3 bomber.py
```
